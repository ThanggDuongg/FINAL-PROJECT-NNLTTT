File này để hướng dẫn sử dụng project
Đầu tiên, để sử dụng project cần cài những thứ như sau:
	JDK, JRE ở đây em sử dụng JDK 16
	IDE: IntelIJ Ultimate 2021.2
	Database: Hệ quản trị csdl Mysql
	Các thư viện em đã để trong folder lib trong project.

Sau khi cài đặt và cấu hình các thứ trên, ta cần import data từ file em đã
để trong project file có đuôi .sql vào MySQL.
Và không quên include các thư viện hỗ trợ (Em đã include vào project, có lẽ
vẫn sẽ còn).
